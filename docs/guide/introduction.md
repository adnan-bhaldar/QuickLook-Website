# Introduction

<span class="ql-badge">Getting Started</span>

## What is QuickLook?

QuickLook brings the beloved **macOS Quick Look** feature to Windows. On macOS, you can press `Space` on any file in Finder to get an instant preview — without opening any application. QuickLook does exactly that for Windows Explorer.

> "The single most productivity-boosting feature I miss from macOS — now available on Windows." — Community user

## Why QuickLook?

Windows doesn't have a native quick-preview mechanism. To view a file, you must open it in its associated application — which can take seconds or minutes. QuickLook eliminates that friction entirely.

## How It Works

QuickLook runs silently in the **system tray**. When it detects you've pressed `Space` in Windows Explorer with a file selected, it intercepts the keypress and renders a live preview in a floating window.

The preview closes when you press `Space` again or `Esc`. If you want to actually open the file, press `Enter`.

## Who Is It For?

QuickLook is built for anyone who browses files regularly:

- **Developers** — quickly review code files, diffs, config files, or build artefacts
- **Designers** — preview images, SVGs, fonts and 3D models without opening design tools
- **Writers** — glance at PDFs, Markdown files and ePubs instantly
- **Everyone** — stop waiting for apps to load just to check what a file contains
