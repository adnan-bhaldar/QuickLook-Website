# Quick Start

<span class="ql-badge">Getting Started</span>

## You're already set up

After installation, QuickLook runs automatically in the background. You'll see its icon in the **system tray** near the clock.

## Your first preview

1. Open **Windows Explorer** (`Win + E`)
2. Navigate to any folder with files
3. Click a file to select it
4. Press **`Space`**

A preview window appears instantly. That's it!

## Customising QuickLook

::: warning Still in Development
The Settings panel is not yet available. This feature is currently being worked on and will be included in a future release.
:::

**Right-click the tray icon** to access:

| Option | Description |
|---|---|
| **Settings** | Remap hotkeys, change UI theme, manage plugins |
| **Pause** | Temporarily disable the Space hotkey |
| **Plugins** | View, enable/disable or remove installed plugins |
| **About** | Version info and links |

## Changing the Hotkey

::: warning Still in Development
The Settings panel is not yet available. This feature is currently being worked on and will be included in a future release.
:::

If `Space` conflicts with another app:

1. Right-click the tray icon → **Settings**
2. Click the key binding under **Hotkey**
3. Press your preferred key or combination (e.g. `Alt+Space`)
4. Click **Save**

::: tip Popular alternatives
`Alt+Space` — avoids conflicts with most apps  
`Ctrl+Space` — common in developer tools  
`Q` — minimalist, single-key approach
:::
