# Features Overview

QuickLook brings powerful file preview capabilities to Windows with an intuitive interface and extensive format support.

## Core Features

### ⚡ Instant Preview

Press the spacebar to instantly preview files without opening them. QuickLook appears in milliseconds, making file browsing incredibly fast.

**Key Benefits:**
- Zero wait time — previews appear instantly
- No application launching overhead
- Minimal system resource usage
- Smooth animations and transitions

### 🎨 Beautiful Interface

Clean, modern interface that matches Windows design language while providing powerful preview capabilities.

- Adapts to light and dark Windows theme
- HiDPI and 4K display support
- Multi-monitor aware with per-display DPI scaling
- Resizable, draggable preview window

### 📁 Advanced File Support

Comprehensive support for virtually every file type you'll encounter.

## Advanced Features

### Zoom & Navigation

- **Zoom in/out** with `Scroll`
- **Reset zoom** with `Ctrl 0`
- **Full screen** with `F11`
- **Navigate images** in the same folder with `←` `→`
- **Navigate PDF pages** with `↓` `↑` or `PgDn` `PgUp`
