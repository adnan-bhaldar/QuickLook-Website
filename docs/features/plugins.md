# Plugins

<span class="ql-badge">Features</span>

QuickLook's plugin system lets the community extend preview support to any file type. Plugins install with a single double-click.

<PluginCard />

## Installing a Plugin

1. Download the `.qlplugin` file from the plugin's GitHub Releases page
2. **Double-click** the `.qlplugin` file
3. QuickLook will prompt you to install — click **Yes**
4. Right-click the tray icon → **Quit**, then relaunch QuickLook

The new format is now supported instantly.

## Managing Plugins

::: warning Still in Development
Plugin management via a Settings panel is not yet available. To remove a plugin, delete its folder from `%AppData%\QuickLook\Plugins`.
:::
