# System Requirements

<span class="ql-badge">Resources</span>

## Minimum Requirements

| Component | Requirement |
|---|---|
| **Operating System** | Windows 10 version 1803 (April 2018 Update) |
| **Architecture** | x64 or ARM64 |
| **RAM** | 200 MB available |
| **Disk Space** | ~20 MB |
| **.NET Runtime** | Bundled — no separate installation |
| **Display** | Any resolution; HiDPI/4K supported |

## Recommended Setup

| Component | Recommendation |
|---|---|
| **OS** | Windows 11 22H2 or later |
| **Architecture** | x64 |
| **RAM** | 500 MB available |
| **Display** | 1080p or higher |

## Windows 11

QuickLook is fully tested and supported on Windows 11, including ARM64 devices like Surface Pro X and Copilot+ PCs.

## Additional Codec Requirements

Certain preview types require optional Windows codecs:

| Format | Codec |
|---|---|
| HEIC / HEIF | [HEIF Image Extensions](https://www.microsoft.com/store/apps/9pmmsr1cgpwg) (Microsoft Store) |
| AV1 Video | [AV1 Video Extension](https://www.microsoft.com/store/apps/9mvzqvxjbq9v) (Microsoft Store) |
| VP9 Video | Included in most Windows 10/11 builds |

::: tip
These codecs are free from the Microsoft Store and install in seconds. QuickLook will prompt you if a required codec is missing.
:::
